# Blog

我的个人博客 -- 主站前端

主要使用`react`搭建而成

请看:eyes:！

## Usage

```
git clone https://github.com/Linn3a/myblog.git
cd myblog
yarn install
yarn run dev
```
会默认在`localhost:5173`运行
